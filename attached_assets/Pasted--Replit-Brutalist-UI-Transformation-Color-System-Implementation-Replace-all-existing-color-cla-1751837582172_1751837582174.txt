## Interactive Enhancements

### Hover Effects

Add these classes to interactive elements:

```css
.interactive-card {
  transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

.interactive-card:hover {
  transform: translateY(-4px);
  box-shadow: 0 20px 40px rgba(49, 47, 44, 0.2);
}

.brutalist-button {
  transition: all 0.2s ease;
  transform: translateY(0);
}

.brutalist-button:hover {
  transform: translateY(-2px);
  box-shadow: 0 8px 16px rgba(210, 247, 90, 0.3);
}

.brutalist-button:active {
  transform: translateY(0);
}
```

## Global Layout Changes

1. **Remove all existing blue color classes** (bg-blue-600, text-blue-600, etc.)
1. **Replace all rounded corners** with sharp edges (remove rounded-* classes)
1. **Increase border widths** from 1px to 2-4px using border-2, border-3, border-4
1. **Convert all text to uppercase** for headers and buttons
1. **Add letter-spacing** (tracking-wide, tracking-tight) to all text
1. **Implement hover transforms** on all interactive elements
1. **Use accent colors only** for status indicators and active states

## Font Weight Hierarchy

- **900 (Black):** Page titles, mega headers
- **800 (Extra Bold):** Section headers
- **700 (Bold):** Card titles, buttons
- **600 (Semi Bold):** Navigation links
- **500 (Medium):** Body text, descriptions
